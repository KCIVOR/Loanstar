Branded action button with 6 variants and 3 sizes, matching LoanStar's financial-software tone (sharp 4px radius, confident primary blue).

```jsx
<Button variant="primary" size="md">Apply for Loan</Button>
<Button variant="secondary">View Schedule</Button>
<Button variant="outline">Download PDF</Button>
<Button variant="danger">Withdraw</Button>
<Button variant="primary" loading>Processing…</Button>
```

Variants: `primary` (solid blue, main CTA), `secondary` (tinted blue), `outline` (blue border), `ghost` (text-only, cancel/tertiary), `danger` (red, destructive), `success` (green, confirm payment/approve). Use `fullWidth` for form CTAs stacked in a narrow column.
