# Loan Application

4-step borrower application flow: Personal Info → Loan Details (amount slider + live 1.5%/mo calculator) → Documents (upload checklist) → Review & Submit (summary + 3 required agreements) → success state with a generated reference code.

Source: `Loan Application.dc.html` in the original mockups.

Files: `LoanApplicationApp.jsx` (`LoanApplicationApp`), `index.html`.
