# Admin Dashboard

Branch-manager overview: 4 KPI cards, a CSS conic-gradient donut (loan status mix), a pipeline progress list by workflow stage/role, and a filterable recent-applications table.

Source: `Admin Dashboard.dc.html` in the original mockups (the bar/line charts there are inline SVG; this port keeps the donut + pipeline bars and drops the two SVG trend charts for brevity — recreate them the same way if needed).

Files: `AdminDashboardApp.jsx` (`AdminDashboardApp`), `index.html`.
