# Staff Operations

Shared workspace for all workflow roles (CSA, CIG, Committee, LRA, Collection) — a role switcher swaps the queue and detail panel, each role sees the same layout with role-specific info fields, document checklist, and action buttons.

Source: `Staff Operations.dc.html` in the original mockups.

Files: `StaffOperationsApp.jsx` (`StaffOperationsApp`), `index.html`.
